.. _Building:

Building
========

macOS & Linux:

.. code:: shell

   git clone git@github.com:doronz88/rpc-project.git
   cd src/rpcserver
   make

iOS (Make sure to have XCode installed):

.. code:: shell

   git clone git@github.com:doronz88/rpc-project.git
   cd src/rpcserver
   ./build_darwin.sh
